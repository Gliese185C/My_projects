Все необходимые файлы и библиотеки для запуска программы находятся в архиве program.rar
Чтобы запустить программу, нужно запустить исполняемый файл Bifid_and_RC4
После запуска, чтобы получить доступ к приложению, нужно ввести логин и пароль
логин - admin
пароль - admin
Оба алгоритма используют ключ для шифрования информации, он находится в правой части приложения
Вверху программы можно выбрать какой алгоритм и декодировать или кодировать информацию
Сильно не критикуйте :)
